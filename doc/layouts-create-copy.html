Copy current layout

If you want to create a custom layout, you do not have to do it from scratch. In fact, it might be easier to choose an already existing layout, have Onboard create a copy of it and apply changes to the copy.

To create a copy of a layout, you can do the following:

  1. Click on the Preferences key in Onboard to open the Onboard Preferences dialogue.

  2. In the Onboard Preferences dialogue click on the Layouts tab and select the layout that you want to copy in order to make it the current layout.

  3. Click on the New button to create a copy of the selected layout. You will be prompted for a name to give to the copy. The files of the newly created copy will be located in the .local/share/onboard/layouts/ directory of your home folder.

  4. At this point, you can switch back to any layout you want to use for Onboard, close the Onboard Preferences dialog and start editing the layout files that were just created.
